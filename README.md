**Project Plans**
<a name="plans"></a>

- Update the Readme as the project is ongoing
- Develop a strategy for organizing content around photography, development, design, resume, and finding ways to just make it...fun
- Utilize BEM to take years of knowledge and make it second-hand and natural
- Utilize SASS to improve file readability
- Build Home, About, Work, Resume, and Contact, Privacy Policy pages
- Integrate GTM and utilize Partytown
- Deploy on Netlify
- Host images on Bunny CDN
- Get as close to 100% Google Speed Score as possible
- (Possibly) consider switching to Tailwind CSS
- Start a blog and eventually create a headless CMS
- Have conditional logic in the contact form
- Integrate Barba JS into the site for faster transitions between pages
